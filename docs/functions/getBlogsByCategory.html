Function getBlogsByCategory

指定されたカテゴリのブログ記事を取得します。

Example

const { contents, totalCount } =
    await getBlogsByCategory(
      'category-id-123',
      1,
      10,
    );
  console.log(
    'カテゴリ内ブログ数:',
    totalCount,
  );
  contents.forEach((blog) => {
    console.log(blog.title);
  });
